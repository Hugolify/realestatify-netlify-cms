## Install

Edit `config/_default/module.yaml` to install the `realestatify-netlify-cms` module with `hugolify-netlify-cms`:
```yml
imports:
  - path: github.com/hugolify/realestatify-netlify-cms
  - path: github.com/hugolify/hugolify-netlify-cms
```
